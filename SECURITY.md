# Security Policy

Hermeneia is in early planning/scaffolding mode.

## Reporting Security Issues

Please do not publicly disclose security vulnerabilities before maintainers have had a chance to review them.

For now, open a private communication channel with the maintainer or create a GitHub security advisory once enabled.

## Sensitive Data

Do not commit:

- API keys,
- OAuth tokens,
- platform credentials,
- private content calendars,
- unpublished client data.

Publishing integrations will require additional security documentation before production use.
